## 根因诊断

### 硬件
- 显卡：**AMD 890M 集成显卡**（Strix Point APU）
- VRAM：**跟系统内存共享**（不是独立）
- 系统内存：121Gi 总 / 22Gi used / 99Gi available
- iGPU 带宽远低于独显（PCIe 通道有限）

### 之前 09:55 误判
- V 说"4 个 model 并存吃 VRAM"
- 实际 `ollama ps` 显示 **0 个 model loaded**（Ollama 启着但都没装进内存）
- 即使用 4 个 model 一起跑，**卸载/装入** 一次就几十秒，跑推理时只装 1 个
- **R1 70B 慢 = 装入后推理时 iGPU 算力不够**（39.6GB 模型权重读取 + KV cache 全在内存带宽限制下）

### 之前 bench 4.57 t/s 数据
- 推测是**独显环境**测的（可能 3080/4090/Radeon 独显）
- 890M iGPU 跑 70B = 性能降级 8-10 倍符合预期

### 发现的问题及解决
1. **libvirt NAT 网桥 virbr0 状态 DOWN** → 重启 libvirtd 后自动恢复 UP
2. **LIBVIRT_FWI 规则阻止入站连接** → 手动插入放行规则
3. **NLA 导致 xfreerdp 连接失败** → 用 `/sec:tls` 绕过
4. **FreeRDP 3.x 语法不兼容旧版** → winapps 生成 1.0 语法命令，需加参数
5. **DISPLAY 环境变量缺失** → ~/.bashrc 加 `export DISPLAY=:0`

### router-agent.js 架构

```
用户消息 → classifyIntent() → 优先级分类
                              ↓
  memory_store > task_create > search > conversation
                              ↓
                      routeIntent() → bus.emit()
                              ↓
                    skill-handler.js 内置 Handler
                              ↓
                      返回 { intent, skill, success, response, data }
```

### CLI 模式

```bash
node tools/router-agent.js --json "记住我喜欢苹果"
node tools/router-agent.js --json "新建任务写周报"
node tools/router-agent.js --json "搜索 AI 最新动态"
```

### 意图优先级

1. `memory_store` — 记住/存储/保存
2. `task_create` — 创建任务/待办
3. `search` — 搜索/调研/新闻
4. `conversation` — 问候/简单问答（不处理）

### Hook 集成

- 配置：`hooks.internal.enabled: true`
- Hook：`router-agent-hook`（message:preprocessed）
- 职责：拦截 structured 意图（search/memory/task），直接返回结果
- 配置已写入 openclaw.json，重启后生效
